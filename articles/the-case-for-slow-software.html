The Case for Slow Software

23 June 2026

We have learned to treat speed as an unqualified good. Faster startup, faster builds, faster everything. But some of the software I trust most is the software that asks me to wait a beat before doing something irreversible.

This is placeholder text for a piece still taking shape. The argument, roughly, is that deliberate pauses — confirmations, previews, undo windows — are not latency to be optimised away but care made legible.
